 [ 狗, 鹿]
 说明 : 攻击力极弱.
 物品 : 肉, 银元, 天桃
 地带 : 新手村附近

 [ 牛, 马 ]
 说明: 攻击力很弱的动物. 用狗、鹿修炼不爽不妨打
       牛和马试试。
 物品: 肉, 皮, 银元, 天桃
 地带: 新手村附近的地区

 [ 犀牛 ]
 说明 : 适合中级玩家修炼的怪物.小看它可会吃大亏哟。
 物品 : 肉, 银元, 天桃
 地带 : 村子东南角旷野.

 [ 老虎 ]
 说明 : 适合高手修炼。新手村中最强的怪物。能够轻
        而易举将它搞定，预示着你有资历闯荡武林啦。
 物品 : 肉, 皮, 银元, 老虎指甲,天桃
 地带 : 村子西南角

 [ 熊 ]
 说明 : 新手村顶级怪物，比起修炼更有利于获取物品。
 物品 : 皮, 银元, 熊掌, 金元, 墨铁矛等
 地带 : 村子朝西

 [ 犀牛王/犀牛王后 ]
 说明 : 新手村顶级怪物，比起修炼更有利于获得物品。
 物品 : 肉, 犀牛角, 金元, 墨铁槌, 墨铁手套 等
 地带 : 村子向南

 [ 白老虎 ]
 说明 : 新手村顶级怪物，比起修炼更有利于获取物品。
 物品 : 肉, 皮, 白老虎指甲, 金元, 墨铁剑 等
 地带 : 村子西南角
